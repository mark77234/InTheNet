# InTheNet 

### OFFLINE
상대방의 Net를 맞추면 이기는 게임 ( 2 player ) (개발중)

### Tutorial
1. 상대방의 Net를 공으로 맞춰야 한다
2. 자신이 맞고 상대 방의 Net 뒤쪽을 맞추면 상대방의 승리 (본인 Net 뒤쪽도 마찬가지)
3. 1 player: 방향키 ⬆⬅⬇⮕ / 2 player: WASD
